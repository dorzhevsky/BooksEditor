С помощью данных приложений и скриптов запускаются юнит-тесты в unix-окружении под mono и строится html-отчет по ним.
1. Положить содержимое папки bin/Release проекта Test в папку на unix-машине
2. В ту же папку положить папку MonoEnv 
3. Выствить права на исполнение нужным файлам: chmod 755 ./nunit-results.exe ./test_runner.sh ./nunit-console4
4. Для запуска юнит-тестов и построения отчета по результатам запустить скрипт test_runner.sh
